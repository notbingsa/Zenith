Zenith is the undisputed *king* of game websites.

## Fullscreen Mode
We have made a fullscreen mode which has support for Smoothwall, which doesn't have a native fullscreen feature. It enlarges whatever game you are playing to fill the screen for maximum comfortability so you can enjoy your time.

## Encryption
We have encrypted our links to avoid detection, so it makes it more difficult for blockers to block Zenith.
